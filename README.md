# Portafolio de Isaac González

Portafolio web personal desarrollado con HTML, CSS y JavaScript, presentando mi experiencia profesional, proyectos y habilidades técnicas.

## Descripción

Este portafolio es una página web responsiva que muestra mi trayectoria profesional en atención al cliente y soporte técnico en empresas como PayPal y OpenTable, así como mi formación académica en Ingeniería en Diseño de Software y Redes.

## Características

- Diseño responsivo y moderno
- Interfaz con estética retro utilizando fuentes personalizadas (Press Start 2P, VT323)
- Gradiente de fondo dinámico
- Secciones organizadas: Introducción, Sobre mí, Proyectos, Experiencia y Contacto
- Formulario de contacto integrado
- Enlaces directos a LinkedIn y descarga de CV

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica del contenido
- **CSS3**: Estilos, gradientes y animaciones
- **JavaScript**: Interactividad
- **Google Fonts**: Tipografías Press Start 2P y VT323

## Estructura del Proyecto

```
Portafolio/
│
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── script.js           # Scripts JavaScript
├── icons/              # Iconos e imágenes decorativas
├── resources/          # Recursos (CV, foto de perfil)
└── README.md           # Documentación del proyecto
```

## Instalación y Uso

1. Clona o descarga este repositorio:
```bash
git clone [URL-del-repositorio]
```

2. Abre el archivo `index.html` en tu navegador web preferido.

**Nota**: Para visualización local completa, se recomienda usar un servidor local como Live Server en VS Code debido a las rutas absolutas de recursos.

## Secciones

### Introducción
Presentación personal con foto de perfil y enlaces de contacto directo.

### Sobre mí
Información personal, formación académica e intereses profesionales.

### Proyectos
Muestra de proyectos desarrollados con las tecnologías utilizadas.

### Experiencia
Historial profesional detallado con responsabilidades y logros en cada posición.

### Contacto
Formulario de contacto para comunicación directa.

## Personalización

Para personalizar este portafolio con tu información:

1. Edita `index.html` con tus datos personales
2. Reemplaza las imágenes en `/resources/` con tu foto de perfil y CV
3. Modifica los colores en `styles.css` según tu preferencia
4. Actualiza los enlaces de redes sociales
